## 1. Launch healthKERI RACK and Mirth Connect

Launch two instances of <Tooltip tip="Routing, Authentication and Confidentiality with KERI">RACK</Tooltip> and two instances of Mirth Connect using the following Docker compose command:

```bash Docker Compose
docker compose -f ./examples/mirth-connect-rack-compose-sample.yaml up
```
Copy cesr files out of docker containers with

```bash Docker
docker cp examples-rack-1-1:/opt/rack/data/Rack1-core.cesr ./rack-1.cesr
docker cp examples-rack-2-1:/opt/rack/data/Rack2-core.cesr ./rack-2.cesr
```
